Imagining Code

Eén dag vol uitdagende workshops rond visual coding, VR en physical computing

23 februari 2019 (9u30 — 18u), Sint Lucas Antwerpen

Van Schoonbekestraat 143, 2018 Antwerpen

Gepassioneerd door technologie en visuele creatie?

Op zaterdag 23 februari 2019 kan je kijken hoe het er écht aan toe gaat. Op die dag organiseert Sint Lucas Antwerpen 3 uitdagende workshops voor 16 tot 18-jarigen.

  • Virtual Reality

    Ontdek de mogelijkheden van VR en maak je eigen werelden. Bouw nieuwe creaties via de Oculus Rift.

  • Physical Computing

    Integreer digitale creaties met de echte wereld. Stuur intelligente machines via je smartphone.

  • Visual Coding

    Maak interactieve visuele creaties via code. Leer de kracht van Processing en creatief programmeren.

Je hoeft zelfs niet te kiezen: gedurende de dag kan je alle 3 workshops volgen. Zo proef je van het hele aanbod. Iedere workshop duurt ongeveer 2 uur.

Studio Data

Vanaf academiejaar 2019-2020 start Studio Data, een nieuwe bachelor-opleiding op Sint Lucas Antwerpen waar we kunst, ontwerp en code combineren.

We organiseren de Imagining Code workshop als een voorproefje op wat je binnen Studio Data allemaal zal tegenkomen. Als je geboeid bent door de workshop, lees dan zeker meer over Studio Data!

Meet our new campus

Begin februari 2019 verhuist Sint Lucas naar een gloednieuwe campus aan de Van Schoonbekestraat in Antwerpen. Ontdek mee hoe deze nieuwe campus een plek is waar je je creativiteit tot uiting kan laten komen!

Time to shape the future

Imagining Code is een fantastische introductie in het creatief gebruik van nieuwe technologie: visual coding, VR, en physical computing.
